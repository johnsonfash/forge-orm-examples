# 03 · DuckDB analytics in the browser

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/johnsonfash/forge-orm-examples/tree/main/03-duckdb-analytics-browser)

Aggregating 5,000 sales rows in the browser with DuckDB-wasm. Columnar storage + vectorised execution — fast even in a tab. Swap the URL to a native `duckdb:` URI and the same code runs server-side.
